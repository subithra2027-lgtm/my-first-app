# 🎓 Academic Analytics Enterprise Portal

## 📌 Project Overview
This project is developed using Python and Streamlit for analyzing student academic performance. It provides an interactive dashboard to monitor attendance, marks, toppers, pass percentage, and subject-wise performance.

## 🚀 Features
- Secure Login System
- Student Performance Dashboard
- KPI Cards
- Interactive Student Database
- Section-wise Topper Analysis
- Pass vs Fail Donut Chart
- Subject-wise Performance Analysis
- Student Performance Tracking
- Regional School Benchmarking
- Download Student Report as CSV

## 🛠 Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Plotly

## 📂 Project Files
- app.py
- requirements.txt
- README.md

## ▶️ Run the Project

Install the required libraries:

pip install -r requirements.txt

Run the application:

streamlit run app.py
